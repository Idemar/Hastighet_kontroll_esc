# Test ESP8266 med motor kontroller

Prosjekt lage motor kontroller for børsteløs RC motor.

I stedenfor å kontrollere motoren via radio kontroll som vanlig.
Ønsket jeg å ha en stasjonær kontroller. Denne motoren skal aldri i luften, det er i hvertfall planen :-)

Eneste jeg trenger er å kunne kontrollere hastigheten så den er konstant (eller justere ved behov i test perioden) Denne skal justeres automatisk i produksjons versjon.
Og siden jeg har flere ESP8266 ligende i en skuff, så hvorfor ikke bare bruke det jeg har.

Jeg bruker vanlig ESC kontroller for motoren. Jeg vil bruke ESP8266 og potensiometeretere hastigheten.

Enkelt lite program, men den gjøre jobben.


Blir spennende å se når jeg setter dette sammen med hoved prosjektet å se hvodan alt virker sammen.
